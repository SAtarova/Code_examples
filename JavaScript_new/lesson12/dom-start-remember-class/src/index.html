Counter Task

Создать схожую верстку: две кнопки и текстовое поле с отображением начального числа счетчика. Реализовать следующий функционал: при нажатии на кнопку "+" увеличивать счетчик на 1, когда счетчик станет равен 10, кнопка должна стать не активной (по нажатию, счетчик больше не увеличивается) и появиться сообщение (alert), что достигнут максимум счетчика, при нажатии на кнопку "-" уменьшать счетчик на 1, если счетчик станет равен 0, вывести сообщение (alert) что достигнут минимум счетчика.

Comment Form Task

Создать схожую верстку: четыре текстовых поля и кнопка отправки комментария. Создать пустой массив, для хранения созданных комментариев. Реализовать следующий функционал: при нажатии на кнопку, проверить чтобы все четыре поля не были пустыми, если они не пустые, сформировать объект со следующими пятью свойствами: id, title, body, rating, author. Свойству id присваивать число, равное длине массива плюс один. После создания объекта, поместить его в конец созданного вами массива с комментариями и очистить все поля. Массив с комментариями вывести при помощи console

Comment Section Task

Создать схожую верстку: карточки с комментариями, а также форму для вывода комментариев, которая содержит: четыре поля и кнопку. Новые комментарии добавляются в конце уже созданных комментариев. Форма создания комментариев дублирует функционал предыдущего задания, с добавлением следующего, нового функционала: если строка с сообщением (свойство body) больше 55 символов, обрезать ее и вставить в конце три точки; провести проверку поля автора, если комментарий с таким же автором уже был написан, изменить рейтинг доверия на "5".

News Task

Скачать предоставленный файл с массивом новостей, вывести новости на экран и реализовать следующий функционал: сортировку новостей по названию, фильтр новостей по имени автора, удаление всех новостей по нажатию на кнопку. Accordions Task

Создать схожую верстку: Четыре аккордеона (общий блок, внутри него: блок с вопросом и блок с ответом). Реализовать следующий функционал: изначально все вопросы открыты. В начальном положении вопросы должны быть "закрыты", стрелочка направлена вправо, по нажатию на вопрос или стрелочку вопрос открывается (ответ появляется снизу), стрелочка направлена вниз. *Класс поворота стрелочки необходимо написать самостоятельно. Стили для аккордеона можно скачать по этой ссылке или найти в исходниках в папке "css -> components -> accordion".

todo list

Создать схожую верстку. У вас есть кнопка создания новой заметки и кнопка "удалить все". По нажатию на кнопку "удалить все" - удаляются все элементы списка. Рядом с каждой заметкой есть три кнопки: удалить, редактировать, готово. Кнопка "удалить" - удаление текущей заметки; Кнопка "редактировать" - редактирование текущей заметки, повторное нажатие сохраняет изменения; Кнопка "готово" - зачеркивает текущую заметку и делает остальные кнопки данной заметки неактивными, повторное нажатие возвращает поле в исходное состояние

Open Comment

При повторном нажатии на окно, комментарий закрывается. Класс для "открытия" модального окна "displayShow"
